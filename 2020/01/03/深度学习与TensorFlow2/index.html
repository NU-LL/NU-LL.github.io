    深度学习与TensorFlow2

    一、TensorFlow2数据类型

    • int、float、double

      • 创建

    • bool

    • string

    tensor常见属性：

    • .device：string，返回当前tensor所在设备的名字
    • .gpu()：tensor，返回一个在gpu上的新的tensor（用于cpu和gpu之间的切换）
    • .cpu()：tensor，返回一个在cpu上的新的tensor（用于cpu和gpu之间的切换）
    • .numpy()：numpy，将tensor转换为numpy类型（numpy肯定在cpu上）
    • .ndim：返回数据的维度信息
    • .shap：返回shap
    • .is_tensor()：判断数据类型是否为tensor
